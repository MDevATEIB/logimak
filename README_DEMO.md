## 🚀 Comment utiliser la démo ?

### 1️⃣ Lancer la démo
Cliquez sur le lien : **[demo.logimak.com](https://VOTRE-NOM.github.io/logimak-demo/)**

### 2️⃣ Charger les données exemple (optionnel)
Cliquez sur le bouton **📦 Charger données exemple** dans la bannière pour avoir des données de test.

### 3️⃣ Explorer les fonctionnalités
- Naviguez dans les différentes sections
- Créez des produits
- Effectuez des ventes
- Générez des rapports
- Imprimez des factures

### 4️⃣ Réinitialiser (si besoin)
Cliquez sur **🔄 Réinitialiser** pour repartir de zéro.

> **💡 Note :** Vos données sont stockées localement dans votre navigateur et ne sont jamais envoyées sur un serveur.

## ✨ Fonctionnalités principales

### 🏪 Gestion du stock
- Ajout illimité de produits
- Organisation par catégories
- Gestion des quantités et alertes de stock faible
- Codes-barres et descriptions détaillées
- Prix unitaires et valeur totale du stock

### 💳 Système de ventes
- Panier intelligent avec calcul automatique
- Génération de numéros de facture uniques
- Informations client (nom, téléphone)
- Historique complet des ventes
- Réimpression de factures

### 📈 Rapports et statistiques
- Rapport des ventes (par période)
- Rapport du stock (inventaire complet)
- Rapport du chiffre d'affaires
- Filtres par jour, mois, année ou période personnalisée
- Export et impression professionnelle

### ⚙️ Configuration
- Informations complètes de l'entreprise
- Logo personnalisé
- Coordonnées de contact
- Numéro fiscal
- Informations additionnelles

### 💾 Sauvegarde et restauration
- Export des données en JSON
- Import de sauvegardes
- Réinitialisation sécurisée
- Sauvegarde automatique dans le navigateur

## 🌍 Développé en Afrique, pour l'Afrique

LOGIMAK est conçu spécifiquement pour répondre aux besoins des commerçants et PME africaines :

- 🌐 **100% hors ligne** : Fonctionne sans connexion Internet
